# Advanced Math Library

Mathematical APIs for CC: Tweaked!

## Overview

This project provides a few generic advanced mathematics libraries for CC: Tweaked.  

- [**Quaternions**](https://techtastic.github.io/Advanced-Math/modules/quaternion.html)  

- [**Matrices**](https://techtastic.github.io/Advanced-Math/modules/matrix.html)  

- [**PID Controllers**](https://techtastic.github.io/Advanced-Math/modules/pid.html)  

[Here](https://github.com/TechTastic/Advanced-Math) is the actual repository of this project.

## Installation

If you want to add all libraries to your server or world, grab the datapack from the following sites:  

- [**CurseForge**](https://www.curseforge.com/minecraft/data-packs/advanced-math)  

- [**Modrinth**](https://modrinth.com/datapack/advanced_math)  

If you want specific library files, simply copy them into your in-game computer and, to use them, follow this:
#### Quaternions
`os.loadAPI("path/to/quaternion.lua")`
#### Matrices
`os.loadAPI("path/to/matrix.lua")`
#### PID Controllers
`local pid = require "path/to/pid"`
